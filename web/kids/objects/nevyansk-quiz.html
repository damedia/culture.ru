ВОПРОС №1

При строительстве башни кирпичи проверяли на прочность простым надежным способом — сбрасывая с высоты. В ход шли только те, что не разбивались.
С какой высоты бросали кирпичи для проверки?

  • С ВЫСОТЫ СЕДЬМОГО ЭТАЖА
  • С ВЫСОТЫ ВТОРОГО ЭТАЖА
  • С ВЫСОТЫ ТРЕТЬЕГО ЭТАЖА

Неправильно...

Это не так. Кирпичи длиной 32 сантиметра сбрасывали с 10-метровой высоты (примерно третий этаж), и для строительства башни использовали только те, что остались целыми.
